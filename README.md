# Task Manager App

A simple task management app built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.  
Features include creating, editing, and managing tasks with a clean, responsive UI.

##  Quick Start
```bash
git clone https://github.com/spoorthy-j/Syntecxhub_task_manager.git
cd Syntecxhub_task_manager
npm install
npm run dev
